# ATM Interface

## Description

A Java-based ATM simulation that allows users to deposit money, withdraw money, and check account balance using Object-Oriented Programming concepts.

## Features

* Deposit Money
* Withdraw Money
* Check Balance
* Input Validation
* Encapsulation using Classes and Objects

## Technologies Used

* Java
* OOP Concepts

## How to Run

1. Compile the program:

```bash
javac ATMInterface.java
```

2. Run the program:

```bash
java ATMInterface
```

## Concepts Used

* Classes and Objects
* Methods
* Encapsulation
* Input Validation
